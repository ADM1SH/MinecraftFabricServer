# Minecraft Fabric Server Configuration

## Overview
A repository for managing the configuration and setup of a Minecraft server using the Fabric mod loader. This includes essential files for server maintenance and administration.

## Key Files
- `ops.json`: List of server operators.
- `whitelist.json`: Controlled access list for the server.
- `banned-players.json`: Administrative ban management.
- `world/`: Reference to world data and statistics.

## How to Use
These files are intended to be placed in the root directory of a Fabric-based Minecraft server.
